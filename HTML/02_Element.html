Element Explain

  • The html element defines the whole document.
  • The body element defines the document body
  • The (h1) Element defines a heading
  • The (p) Element defines a paragraph
  • Some HTML elements will display correctly, even if you forget the end tag
  • HTML element with no content are called empty element is empty element without a clolsing tag the (br) element

Use Lowercase Tags

HTML tags are not case sensitive The HTML5 standard does not require lowercase tags, but we recommends lowercase in HTML, and demands lowercase for stricter document types like XHTML.
